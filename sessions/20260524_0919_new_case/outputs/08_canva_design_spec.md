# 08_canva_design_spec

## 基本情報
- サイズ：1600 x 2560 px
- 形式：JPEG（エクスポート時の品質は高め）

## レイヤー構成（推奨）
1. 背景（グラデーション）
2. メインイラスト（小さな植物・ノート等）
3. タイトルテキスト（上部）
4. 装飾アイコン（必要に応じて薄く）
5. 下部キャッチコピー帯（半透明の矩形）
6. キャッチテキスト

## テキスト設定（推奨値）
- タイトル：フォント例「Merriweather」「Noto Serif JP（中太）」など。サイズ目安：72〜120pt（調整必須）
- キャッチ：フォント例「Noto Sans JP」「Hiragino Kaku Gothic」。サイズ目安：28〜48pt
- 強調語句：アクセントカラーで着色、または太字適用

## 配色（参照）
- 背景：#FFF7F3 → #FFFBF8 の穏やかな縦グラデーション
- アクセント：#FFDDD2（ピンク）、#CFE8D5（グリーン）
- テキスト濃色：#3B3B3B

## イラストの指示
- スタイル：写実でない、やさしい手描き風。線は細め、塗りはソフト。
- 配置：タイトルを邪魔しない位置（例：左下寄せ、または背景に薄く敷く）
- 透過PNGまたはSVGで読み込み、サイズと不透明度を調整する

## レイアウトルール
- タイトルは上部に大きく配置（上下の余白を十分に確保）
- キャッチコピーは下部帯に収める（背景とテキストのコントラストは十分に）
- モバイル表示のサムネイル想定で、中心上寄せのタイトルが切れないようにレイアウト

## 書き出し設定
- JPEG品質：90〜95％
- カラープロファイル：sRGB

## Canvaで使う検索キーワード（素材探し）
- "hand drawn leaves" "notebook illustration" "pastel gradient background" "soft line art"

## 備考
- キャッチコピーの最終決定により下部帯の高さ・フォント量を調整してください。
