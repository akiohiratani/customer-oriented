# ステップ3：ターゲットの状況と心理を考える

## 1. このステップの目的

案件文に書かれている情報をもとに、表紙を見る人の状況と心理を仮説として整理する。

ここでのターゲット心理は、実際の読者調査ではない。案件文・参考画像・ここまでの分析から作る仮説である。

## 2. 案件文に書かれているターゲット情報

### 事実

- メインタイトルに「初心者に優しい」とある
- メインタイトルに「投資はじめの一歩」とある
- キャッチコピーに「知識ゼロでも安心して始められる」とある
- キャッチコピーに「無理なくお金を増やす仕組み」とある

### 未記載

- 年齢
- 性別
- 職業
- 投資経験の年数
- 投資対象
- 電子書籍を見る場所・販売場所

### 確認事項

- どの程度の初心者を想定しているか
- NISA、株式、投資信託など具体的な投資対象を出してよいか
- 初心者マークをそのまま使う必要があるか

## 3. ターゲットの状況仮説

### どこで見るか

事実：
制作物は電子書籍の表紙である。

仮説：
電子書籍ストア、SNS告知、販売ページ、スマートフォン画面などで縮小表示される可能性がある。

確認事項：
実際の販売場所は未記載。

### どんな気分で見るか

事実：
「初心者に優しい」「知識ゼロでも安心」と書かれている。

仮説：
投資に興味はあるが、難しそう・損しそう・何から始めればよいか分からない、という不安を持っている可能性が高い。

確認事項：
読者の具体的な悩みは未記載。

### 何に困っている可能性があるか

事実：
タイトルは「投資はじめの一歩」である。

仮説：
投資を始める前段階で、専門用語や始め方の分からなさに引っかかっている可能性がある。

確認事項：
投資対象や本の中身の具体章立ては未記載。

### 何に魅力を感じる可能性があるか

事実：
キャッチコピーに「無理なくお金を増やす仕組み」とある。

仮説：
大きく儲かることよりも、無理なく、安心して、お金を増やす考え方に魅力を感じる可能性がある。

確認事項：
収益性をどこまで強く訴求してよいかは未確認。

## 4. 初期草案7案

### 草案A：安心スタート訴求

伝えること：
知識ゼロでも大丈夫。まずは安心して一歩目を踏み出せる。

根拠となる事実：
- 「初心者に優しい」
- 「知識ゼロでも安心して始められる」

想定するターゲット心理：
投資に興味はあるが、難しそうで不安。

弱点：
「お金を増やす仕組み」の期待感がやや弱くなる可能性がある。

### 草案B：仕組み理解訴求

伝えること：
投資は勘やセンスではなく、仕組みを知れば始められる。

根拠となる事実：
- 「無理なくお金を増やす仕組み」
- 「投資はじめの一歩」

想定するターゲット心理：
投資に興味はあるが、何を学べる本なのか分からないと手に取りにくい。

弱点：
言い方を間違えると、少し勉強感が強く見える。

### 草案C：投資入門書訴求

伝えること：
これは投資初心者のための入門書である。

根拠となる事実：
- 「初心者に優しい」
- 「投資はじめの一歩」

想定するターゲット心理：
自分向けの本かどうかを一瞬で判断したい。

弱点：
表現が普通すぎると、数ある入門書に埋もれる。

### 草案D：ハードル低下訴求

伝えること：
投資は難しいものではなく、最初の一歩から始められる。

根拠となる事実：
- 「知識ゼロでも安心」
- ユーザー回答「ハードルが低く見える」

想定するターゲット心理：
投資という言葉だけで身構えてしまう。

弱点：
投資の具体性が弱いと、何の本か分かりにくくなる。

### 草案E：お金を増やす期待訴求

伝えること：
無理なくお金を増やす仕組みを知れる。

根拠となる事実：
- 「無理なくお金を増やす仕組み」

想定するターゲット心理：
投資で将来のお金を少しでも増やしたい。

弱点：
強く出しすぎると、初心者向けの安心感より儲け感が勝ってしまう。

### 草案F：参考画像寄せ実用書訴求

伝えること：
大きな文字とお金のイラストで、実用書らしく分かりやすく伝える。

根拠となる事実：
- 参考画像が黄色・白・黒、太字、大きな文字の実用書風

想定するターゲット心理：
本棚や一覧で目立つ表紙に反応する。

弱点：
参考画像に寄せすぎると模倣感が出る。

### 草案G：読みやすさ訴求

伝えること：
難しい専門書ではなく、初心者が読みやすい投資本である。

根拠となる事実：
- 「初心者に優しい」
- 「知識ゼロでも安心」

想定するターゲット心理：
専門用語だらけの本を避けたい。

弱点：
投資の成果や仕組みの魅力が弱くなる可能性がある。

## 5. 学習ポイント

ターゲット心理は断定しない。

ただし、案件文の言葉から「読者が何を不安に思っているか」「何を見れば手に取りやすくなるか」を仮説として複数出すことで、後続の案比較がしやすくなる。
