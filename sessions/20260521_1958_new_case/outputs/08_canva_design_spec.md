# Canva制作指示書

## 1. この指示書の目的

この指示書は、ステップ3〜5で整理した顧客目的・ターゲット心理・訴求軸・デザイン翻訳をもとに、Canvaで電子書籍表紙を制作できる形へ落とし込むためのもの。

Canva制作では、見た目の好みだけで判断せず、以下を優先する。

1. 顧客目的に合っているか
2. 投資初心者に伝わるか
3. 案件文の制約を守っているか
4. メインタイトルが最も目立つか
5. Canvaで実際に作れるか

## 2. 前提情報

### 案件概要

案件名：
電子書籍「初心者に優しい 投資はじめの一歩」表紙デザイン

制作物：
電子書籍の表紙

用途：
電子書籍の表紙として使用

サイズ：
1,600 x 2,560 ピクセル

納品形式：
JPEG

著者名：
今回は無し

### 必須文言

メインタイトル：
初心者に優しい🔰　投資はじめの一歩

キャッチコピー：
知識ゼロでも安心して始められる

キャッチコピー：
無理なくお金を増やす仕組み

### 顧客目的の仮説

投資に少し興味がある初心者に、難しそうと思わせず、手に取って少しでも読んでもらうこと。

### 採用したコンセプト

投資の不安を下げる、最初の一歩の表紙

## 優先度1デザイン案

## 1. デザイン設定

サイズ：
1,600 x 2,560 px

書き出し形式：
JPEG

背景色：
明るい黄色をベースに、中央に白い吹き出しまたは白い大きなタイトルエリアを置く。

外側の余白：
上下左右 80〜120px 程度を目安にする。

全体の印象：
明るい、親しみやすい、初心者向け、実用書風。

## 2. Canvaでの作業方法

開始方法：
- Canvaで「カスタムサイズ」を選ぶ
- 幅 1,600、高さ 2,560 px を入力する
- 空白デザインから始める
- 必要に応じて実用書・ビジネス書系テンプレートを参考にする
- ただしテンプレートより案件文の必須文言を優先する

## 3. レイアウト方針

### 全体構成

上部：
「初心者に優しい🔰」を黄色または白のラベルに入れる。

中央：
「投資はじめの一歩」を最も大きく配置する。

中央周辺：
白い吹き出しまたは白背景でタイトルを囲い、読みやすさを作る。

下部：
キャッチコピー2本を配置する。

右下または下部：
コイン、上向きグラフ、矢印など投資と分かるイラストを置く。

### 視線の流れ

1. 初心者に優しい
2. 投資はじめの一歩
3. 知識ゼロでも安心して始められる
4. 無理なくお金を増やす仕組み
5. 投資イラスト

## 4. 画像・イラスト配置

使用するイラスト：
- コイン
- 上向きグラフ
- 矢印
- 若葉または初心者マーク風の小さな要素

配置場所：
- コインはタイトル右側または下部
- グラフは下部右側
- 初心者マーク風要素は「初心者に優しい」の近く

注意：
- イラストは3種類程度までに抑える
- 文字の上に重ねない
- 投資対象が未記載のため、NISAや株式など具体名を勝手に入れない

## 5. テキスト要素

### メインタイトル

文言：
初心者に優しい🔰
投資はじめの一歩

役割：
表紙で最も目立つ要素。

優先度：
最優先。

### サブコピー1

文言：
知識ゼロでも安心して始められる

役割：
初心者の不安を下げる。

優先度：
高い。

### サブコピー2

文言：
無理なくお金を増やす仕組み

役割：
読むメリットを伝える。

優先度：
中。

## 6. 文字設定

### メインタイトル

サイズ目安：
「投資はじめの一歩」は全体幅の70〜85%を使う大きさ。

太さ：
極太または太字。

配置：
中央。

色：
黒を基本に、重要語の一部を黄色・金色・緑で強調してもよい。

注意：
キャッチコピーより必ず大きくする。

### サブコピー

サイズ目安：
メインタイトルの35〜50%程度。

太さ：
太字または中太。

配置：
タイトル下または下部帯。

色：
黒、濃い緑、濃い茶色など読みやすい色。

## 7. 配色

背景：
明るい黄色

タイトルエリア：
白

メイン文字：
黒

アクセント：
金色、緑、オレンジを少量

避ける色：
- 暗い紺を広く使う
- 赤を多用する
- ギラギラした金色を多用する

配色理由：
参考画像の黄色・白・黒の実用書風に合わせ、初心者向けの明るさと視認性を両立するため。

## 8. 余白・整列ルール

外側余白：
80〜120px 程度

要素間余白：
文字同士が詰まりすぎないようにする。

文字と画像の距離：
最低30〜50px程度は空ける。

整列：
中央揃えを基本にし、タイトルの読みやすさを優先する。

避けること：
- 文字を斜めにしすぎる
- イラストを多く入れすぎる
- キャッチコピーを小さくしすぎる

## 9. 装飾ルール

使ってよい装飾：
- 吹き出し
- 丸ラベル
- 下部帯
- 矢印
- コイン
- グラフ

使わない方がよい装飾：
- 複雑な金融チャート
- 過剰な札束
- 高級感の強すぎる背景
- 細かすぎる説明文

装飾の目的：
初心者向けの安心感と、投資のテーマを一瞬で伝えること。

## 10. Canva操作手順

1. Canvaを開く
2. カスタムサイズで 1,600 x 2,560 px を指定する
3. 背景を明るい黄色にする
4. 中央に白い吹き出しまたは白い角丸なしに近いタイトルエリアを置く
5. 上部に「初心者に優しい🔰」ラベルを置く
6. 中央に「投資はじめの一歩」を極太で配置する
7. タイトル周辺にコインまたはグラフのイラストを置く
8. 下部に「知識ゼロでも安心して始められる」を配置する
9. その下または別帯に「無理なくお金を増やす仕組み」を配置する
10. 文字サイズと行間を調整する
11. スマホ表示を想定して縮小確認する
12. メインタイトルが最も目立つか確認する
13. JPEGで書き出す

## 11. 制作時の確認ポイント

- サイズ指定を守っているか
- JPEGで書き出せるか
- メインタイトルが最も目立つか
- キャッチコピーがタイトルを引き立てているか
- 投資と分かるイラストが入っているか
- 難しそうに見えないか
- 何を学べる本か分かるか
- 著者名を入れていないか
- 参考画像に寄せすぎて模倣感が出ていないか

## 優先度2デザイン案

## 1. デザイン設定

サイズ：
1,600 x 2,560 px

書き出し形式：
JPEG

背景色：
白を広めに使い、黄色の帯やアクセントで視認性を出す。

全体の印象：
分かりやすい、整理されている、仕組みが学べそう。

## 2. レイアウト方針

上部：
「初心者に優しい🔰」を小さめのラベルで配置。

中央：
「投資はじめの一歩」を大きく配置。

下部：
「無理なくお金を増やす仕組み」を太字の帯で配置。

右側または下部：
コインが積み上がるイラストと上向き矢印を配置。

## 3. 優先度1との違い

優先度1：
安心感を中心にする。

優先度2：
お金を増やす仕組みが分かることを中心にする。

## 4. 注意点

- 「お金を増やす」を強くしすぎない
- 儲け話っぽく見せない
- 黄色・白・黒の実用書風は維持する
- タイトル最優先のルールは変えない

# 優先度1案と優先度2案の比較

| 項目 | 優先度1案 | 優先度2案 |
|---|---|---|
| 強く伝える価値 | 安心して始められる | 仕組みが分かる |
| 向いている心理 | 投資が怖い | 何を学べるか知りたい |
| 顧客目的との整合性 | とても高い | 高い |
| 案件文との整合性 | とても高い | 高い |
| Canvaでの作りやすさ | 高い | 高い |
| リスク | 少し弱く見える可能性 | 儲け感が強くなりすぎる可能性 |
| 採用すべき場面 | 初心者向けを最優先したい場合 | 読むメリットを強めたい場合 |

# Canvaで使うとよい機能

- カスタムサイズ
- 図形
- 吹き出し
- テキストの太字
- 素材検索：コイン、グラフ、矢印、初心者、若葉
- 整列
- グループ化
- JPEG書き出し

# Canvaで避けること

- テンプレートの雰囲気に引っ張られすぎる
- 必須文言を省く
- メインタイトルより装飾を目立たせる
- 文字を小さくしすぎる
- 参考画像をそのまま真似する
- 具体的な投資商品名を勝手に追加する

## 学習ポイント

Canva制作では、見た目を作る前に「なぜこの配置・色・文字サイズにするのか」を説明できる状態にする。

今回なら、メインタイトルを最も大きくする理由は「案件文で指定されているから」であり、黄色・白・黒の太字構成にする理由は「参考画像と初心者向け実用書風の方向性に合うから」である。
