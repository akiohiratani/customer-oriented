## 学習目的

クラウドワークスの実案件を教材にして、案件文・添付画像・希望イメージ・制約条件から顧客志向を学ぶ。

最重要目的は、バナーや表紙を作ることではなく、以下の力を身につけること。

1. 案件文から事実を抜き出す力
2. 顧客が本当に達成したい目的を考える力
3. 広告や表紙を見る人の状況・心理を考える力
4. 想定ターゲットに近い視点で、不満・課題・理想状態を整理する力
5. 複数の草案を比較し、根拠を持って採用案を決める力
6. 訴求すべき価値を優先順位づけする力
7. 顧客目的をデザイン要素へ翻訳する力
8. Canvaで制作できる具体的な形に落とし込む力
9. 制作後に、なぜそのデザインにしたのか説明する力
10. 採用・不採用に関係なく、学びを振り返る力
