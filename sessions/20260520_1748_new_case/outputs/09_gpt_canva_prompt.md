# GPT / Canva相談プロンプト

## 1. Canvaでロゴ案を作るためのプロンプト

以下の条件で、不動産会社の新サービスブランド用ロゴを作成してください。

サービス名は「おうちの売却プラス by 越後ホームズ」です。
別案として「おうちの売却＋ by 越後ホームズ」も検討します。

ロゴは2行構成を基本にしてください。

- 上段：おうちの売却プラス、または、おうちの売却＋
- 下段：by 越後ホームズ

コンセプトは「相談から始まる、家のプラス循環」です。

戸建売却を単なる手放しではなく、相談を通じてお客様・家・地域にプラスの変化を生む入口として表現したいです。

デザイン条件：

- 戸建の家をモチーフにする
- 「＋（プラス）」を自然に取り入れる
- 親しみやすさ、安心感、相談しやすさを感じる
- シンプルで視認性が高い
- 看板、WEB、チラシで長期使用できる
- よくある不動産ロゴに見えすぎない
- センチュリー21を想起させる黒×ゴールド主体は避ける
- 既存ロゴや商標に似たデザインは避ける

推奨表現：

- 家の屋根、窓、玄関、または道の一部にプラスを自然に組み込む
- 丸みのあるゴシック体で、読みやすく親しみやすい印象にする
- グリーン、ブルーグリーン、ミント、やわらかいオレンジなど、安心感と前向きさがある配色にする
- カラー版とモノクロ版の両方で成立する構成にする

避けたい表現：

- 家の屋根と四角だけの汎用的な不動産ロゴ
- テンプレート素材そのままに見えるロゴ
- 装飾が多すぎるロゴ
- 強い営業感があるロゴ
- 子ども向けに見えるかわいすぎるロゴ

## 2. GPTにロゴ案を相談するためのプロンプト

あなたはロゴデザイナーです。
以下のクラウドワークス案件に対して、提案用のロゴコンセプトを3案出してください。

案件概要：

- 不動産会社が新たに「戸建売却」を軸としたサービスブランドを立ち上げる
- サービス名は「おうちの売却プラス by 越後ホームズ」または「おうちの売却＋ by 越後ホームズ」
- 単なる売却ではなく「気軽に相談できる窓口」として、売却にまつわる悩みをワンストップで解決できるブランドを目指している
- 「プラス」には、相談を引き受ける、売却によってプラスの価値や変化が生まれる、地域にもプラスの循環が生まれるという意味がある
- 家モチーフとプラスを自然に取り入れたい
- 親しみやすさ、安心感、相談しやすさ、視認性、普遍性が必要
- よくある不動産ロゴはNG
- センチュリー21加盟店だが、今回のブランドは別ブランドとして差別化したい

各案について、以下を出してください。

1. コンセプト名
2. シンボルの形
3. プラス記号の入れ方
4. フォント方向
5. カラー方向
6. その案が案件文に合っている理由
7. 類似ロゴっぽく見えないための注意点

## 3. 提案文作成用プロンプト

以下のロゴ案について、クラウドワークス提出用の短い提案文を作ってください。

条件：

- ですます調
- 300〜500字程度
- デザイン意図が伝わる
- 案件文のコンセプトに沿っていることを説明する
- 「戸建売却」「気軽に相談できる窓口」「プラスの価値・変化・循環」「長期使用」「視認性」に触れる
- 著作権や類似性について断定的に保証しない
- ただし、汎用的な不動産ロゴに寄りすぎないよう意識したことは説明する

ロゴ案の内容：

- 家モチーフとプラス記号を一体化
- プラスは窓・入口・循環の起点として表現
- 丸みのある読みやすいロゴタイプ
- グリーン系を中心に、安心感と前向きさを表現
- 2行構成で「おうちの売却プラス」と「by 越後ホームズ」を配置
