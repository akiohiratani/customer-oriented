# ステップ6：Canva制作指示書

## 1. 制作目的

60代シニア向けの電子書籍表紙を作成し、メインタイトルを最優先で目立たせつつ、やさしい雰囲気と読みやすさを両立する。

## 2. 制作コンセプト

「見やすさで安心を届ける、60代のためのやさしい仕事ガイド」

## 3. 基本仕様

- サイズ：1600×2560ピクセル
- 形式：JPEG
- 著者名：不要
- ターゲット：60代シニア層、未経験・体力に不安がある人
- 主要文字要素：メインタイトル、キャッチコピー2行
- 参考画像：`sessions/20260523_1655_new_case/assets/` に保存済み

## 4. Canva制作手順 - 参考画像に基づく構成

### 手順1：キャンバスを作成する

- Canvaで新規デザインを作成し、カスタムサイズを `1600 x 2560` に設定する。
- 背景を濃紺またはダークネイビー系 `#1C2C42` または濃いティール系 `#1A4A52` に設定する。

### 手順2：メインタイトルを上部に配置する

- メインタイトル「60代　未経験OK！一番ラクな仕事１０選」を上部に大きく配置する。
- フォント：太めのゴシック系（例：Noto Sans JP Bold、M PLUS Rounded 1c Bold）
- 色：黒 `#000000` または濃いグレー `#2B2B2B`
- 複数行に分けて読みやすくする。
- 「未経験OK」「１０選」を黄色で強調するか、黄色い円形バッジで「月5万円」を囲む。

### 手順3：60代シニアの人物イラストを中央に配置する

- 親しみやすい表情で、やさしい雰囲気のシニア人物イラスト（男性または女性、複数でも可）を中央に配置する。
- イラストは仕事をしているようなポーズまたは前向きなポーズが好ましい。
- 参考画像のように、イラストを背景と明確に分離できるイラスト素材を選ぶ。

### 手順4：下部に帯状のキャッチコピー背景を作る

- 画面下部（全幅）に濃いネイビーブルーまたはティール系の帯を配置する。
- 帯の幅はしっかりとした存在感を持たせる（全高の20〜30%程度）。
- 帯の背景色は濃いネイビー `#1A3A52` または濃いティール。

### 手順5：帯の上にキャッチコピーを配置する

- 1行目「体力に自信がなくても月5万円稼ぐ」を白テキストで配置。
- 2行目「シニアの優しい働き方ガイド」を黄色 `#FFD700` でテキストを配置し、強調する。
- フォント：メインタイトルと同じゴシック系、やや小さめサイズ。
- テキストは帯の内側で十分な余白を取る。

### 手順6：黄色い強調要素を入れる

- 黄色い円形バッジ：「月5万円」「未経験OK」など重要キーワードを囲む。
- 黄色は目立たせつつも、全体のバランスを損なわないよう配置する。
- 黄色の明るさが派手すぎないよう注意（参考画像の黄色の程度を参考にする）。

### 手順7：全体確認と微調整

- 遠目でもタイトル、イラスト、キャッチコピーが明確に判別できるか確認する。
- イラストが背景と浮いて見えるか確認する。
- テキスト全体が小さなサムネイルでも読めるサイズか確認する。

### 手順8：JPEG形式で書き出す

- JPEG形式で書き出し、ファイル名例：`cover_60s_easy_work_1600x2560.jpg`


## 5. 色とフォントの推奨

### カラーパレット

- ベース背景：濃紺 `#1C2C42` または濃いティール系 `#1A4A52`
- メイン文字：黒 `#000000` または濃いグレー `#2B2B2B`
- 強調色：黄色 `#FFD700` または `#FFC700`
- 帯の背景：濃いネイビーブルー `#1A3A52` または濃いティール
- 帯のテキスト：白 `#FFFFFF` （1行目）と黄色（2行目）

### フォント候補

- Noto Sans JP Bold
- M PLUS Rounded 1c Bold
- Rounded M+ 1c
- ヒラギノ角ゴシック系の太めフォント
- いずれも可読性重視で、黒い太い文字に向いた選択

## 6. デザイン注意点（参考画像に基づく）

- 背景は濃い色を優先する（参考画像のように）。
- 黄色で重要メッセージを目立たせる（バッジまたは円形）。
- キャッチコピーは帯状で下部に配置する（参考画像の構成を踏襲）。
- イラストは中央に配置して注目を集める。
- 小さなサムネイル表示でも判別しやすい構成。

- [ ] メインタイトルが黒い太い文字で大きく目立つ。
- [ ] 「未経験OK」「１０選」「月5万円」などが黄色で強調されている。
- [ ] 中央に60代シニアの親しみやすいイラストがある。
- [ ] 下部に濃いネイビーの帯がある。
- [ ] 帯の中にキャッチコピーが白と黄色で配置されている。
- [ ] キャッチコピー1行目が白、2行目が黄色で目立つ。
- [ ] 著者名が入っていない。
- [ ] JPEG形式で出力できる。
- [ ] 参考画像の構成を踏襲している。
- [ ] 遠目でもタイトルが判別しやすい。
