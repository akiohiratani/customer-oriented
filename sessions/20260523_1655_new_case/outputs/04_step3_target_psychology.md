# ステップ3：ターゲットの状況と心理を考える

## 1. このステップの目的

案件文に書かれているターゲット情報をもとに、広告を見る人の状況と心理を仮説として整理する。

このステップでは、最初から1案に絞らず、複数の草案を作成して比較できる状態にする。

## 2. 案件文に書かれているターゲット

### 事実

- 60代のシニア層がターゲットである。
- 未経験OKの仕事を求める人向けである。
- 体力に自信がない人でも月5万円稼げることを伝えたい。
- 「シニアの優しい働き方ガイド」という言葉が入っている。
- 著者名は不要で、文字情報はタイトルとキャッチコピーに集中できる。
- 参考画像が提示されている。

### 未記載

- 性別や具体的なライフステージ（定年後、再就職、主婦など）は明記されていない。
- 色味や具体的なビジュアル表現の好みは明記されていない。
- 表紙に人物を入れるか、抽象的に表現するかは未記載。
- どの程度のやさしさ・上品さが好ましいかは明記されていない。

### 要確認

- 参考画像の中で特に採用したい要素はどれか。
- 対象となる60代のイメージ（男性／女性／共通など）。
- どのような表紙の雰囲気が「ラクな仕事」に最も響くか。

## 3. ターゲットの状況仮説

### どこで見るか

事実：
- 電子書籍の表紙として使用される。サイズは1600×2560ピクセル。

仮説：
- スマホやタブレットの電子書籍一覧、Amazonなどのサムネイル画面で見る可能性が高い。
- そのため遠目でも判別できる視認性が必要である。

確認事項：
- 対象プラットフォームがAmazonか、他の電子書籍サイトか。

### どんな気分で見るか

事実：
- 60代の未経験者、体力に自信がない人を想定している。

仮説：
- 「本当に自分でもできるのか」「自分にふさわしいか」を慎重に判断している可能性が高い。
- 不安や迷いがある一方で、少しでも負担が軽そうな働き方・副収入を探している。

確認事項：
- ターゲットが感じている具体的な不安要素（体力、経験、年齢、収入見込みなど）。

### 何に困っている可能性があるか

事実：
- 体力に自信がない人でも月5万円稼げる仕事を探している。
- シニアの優しい働き方ガイドが求められる。

仮説：
- 体力面の負担が少ない仕事を知らない、あるいは情報が少ないことに困っている。
- 未経験OKと書かれていても、本当に始められるか不安を抱えている。
- 年齢による制約や再就職・副業への懸念がある。

確認事項：
- ターゲットが重視する「ラクさ」の具体的な中身（勤務時間、体力、精神的負担、準備の簡単さなど）。

### 何に魅力を感じる可能性があるか

事実：
- 体力に自信がなくても月5万円稼ぐこと。
- シニアの優しい働き方ガイドという文言。

仮説：
- 「未経験OK」「シニア向け」「優しい働き方」という要素に安心感を感じる。
- 仕事の現実感よりも、無理なく続けられそうなイメージに魅力を感じる。
- 具体的な数字「月5万円」があることで、目標感と現実感が高まる。

確認事項：
- ターゲットが「優しい働き方」に期待する具体的な手触り感。

## 4. ターゲット心理の注意

ここは推論を含むため、断定しない。

案件文にない情報は「仮説」として扱う。

AIが考えたターゲット心理は、実際の市場調査結果ではない。

## 5. 初期草案7案

### 草案A：タイトル主役のシンプル強調

伝えること：
- メインタイトルを最大化して、一目で「60代 未経験OK！一番ラクな仕事１０選」とわかる表紙。

根拠となる事実：
- メインタイトルを最も目立つ形で表現する必要がある。
- 著者名は不要で、文字量を絞れる。

想定するターゲット心理：
- いちばん重要な情報を一瞬で確認したい。
- タイトルの明快さが安心感につながる。

弱点：
- 見た目が単調になりやすく、雰囲気が弱くなる可能性がある。

### 草案B：安心感を出すソフトカラー表紙

伝えること：
- 落ち着いた色調とやさしい余白で、60代が安心して手に取りやすい雰囲気。

根拠となる事実：
- 60代シニア向けであり、安心感や読みやすさが重要になる。
- 参考画像の雰囲気がやさしさを示唆している。

想定するターゲット心理：
- 強すぎない雰囲気で「自分でも読みやすそう」と感じる。

弱点：
- タイトルの主張が弱くなり、遠目では判別しづらくなる可能性がある。

### 草案C：「10選」数字強調の実用感

伝えること：
- 「10選」の数字を強調して、具体的で実践的な内容を期待させる。

根拠となる事実：
- メインタイトルに「一番ラクな仕事１０選」と数字が入っている。
- ターゲットは現実的な成果をイメージしたい可能性がある。

想定するターゲット心理：
- 数字に安心感や信頼感を感じ、内容が整理されている印象を受ける。

弱点：
- 数字強調がややビジネス書的になり、やさしさが薄れる危険がある。

### 草案D：キャッチコピーを軸にした親しみ訴求

伝えること：
- 「体力に自信がなくても月5万円稼ぐ」「シニアの優しい働き方ガイド」を目立たせる。

根拠となる事実：
- キャッチコピーがターゲットに寄り添う具体的な言葉を持つ。
- タイトルより目立たない形で補強する指示がある。

想定するターゲット心理：
- 自分の不安や希望に直接答えてくれる本だと感じる。

弱点：
- タイトルとのバランスを間違えると、主訴求がぼやける可能性がある。

### 草案E：ほっとする「やさしさ」と「頼れる仕事」

伝えること：
- 60代が安心して読める、やさしさと頼もしさを両立した表紙。

根拠となる事実：
- シニアの優しい働き方ガイドという言葉。
- 体力に自信がなくても稼げるという具体的なベネフィット。

想定するターゲット心理：
- 気負いなく始められそうな安心感を感じる。

弱点：
- 表現が抽象的になりすぎると、何の本か伝わりにくい。

### 草案F：高齢者でも見やすい大きな文字と余白

伝えること：
- 読みやすさを最優先し、大きな字と広い余白で構成する。

根拠となる事実：
- 60代のターゲットと電子書籍サムネイル表示を考慮する必要がある。

想定するターゲット心理：
- 本の内容がやさしく、読みやすいものであると感じる。

弱点：
- 見た目がシンプル過ぎてインパクト不足になる可能性がある。

### 草案G：未来志向のプラス感と前向きさ

伝えること：
- 仕事を通じて「前向きな変化」「プラスの循環」が生まれるイメージ。

根拠となる事実：
- 10選の中からラクな仕事を見つける明るさ。
- やさしい働き方という文脈。

想定するターゲット心理：
- 生活を少し豊かにする希望を感じる。

弱点：
- あまりに抽象的だと、内容がぼやけて見える可能性がある。
