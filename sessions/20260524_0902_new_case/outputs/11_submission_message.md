# コンペ提出文

はじめまして。

本案件の目的を踏まえ、家族持ちの主婦が「スマホ1本でやさしく動画編集を始められる」と直感できる表紙を重視してデザインいたしました。

今回のデザインでは、メインタイトル「スマホ１本！超やさしい動画編集術」を大きく上部に配置し、手軽さと安心感が伝わるよう構成しています。

また、下部のキャッチコピー帯には「子どもの成長記録や趣味の発信に」「ちょっとした副業にもなる！」を載せ、家族の記録と副業の両方へ訴求するバランスを意識しました。

ターゲットが家族持ちの主婦である点を踏まえ、淡く明るいパステル調の色味と、写実的でないやさしいタッチのイラストを使い、親しみやすさと読みやすさを意識しています。

メインコピーの視認性を最優先しつつ、サブコピーが自然に補強するように調整しました。

修正のご要望がございましたら、文言・色味・配置の調整に対応いたします。

ご検討のほど、よろしくお願いいたします。
