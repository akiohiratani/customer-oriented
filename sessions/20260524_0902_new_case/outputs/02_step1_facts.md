# ステップ1：案件文から事実を抜き出す

## 1. このステップの目的

案件文から、デザイン判断に必要な事実を抜き出す。好みや思いつきでデザインを決めるのではなく、顧客目的・ターゲット・制約条件・必須要素に関わる事実を先に整理する。

## 2. ユーザーが選んだ重要事実

ユーザーによる明示的な選択はありませんでしたが、本ステップでは以下の事実を重要と判断しました。

1. サイズは 1,600 x 2,560 ピクセル、ファイル形式はJPEG。
2. 著者名は表紙に入れない。
3. メインタイトル「スマホ１本！超やさしい動画編集術」を最も目立つ形にする。
4. ターゲットは家族持ちの主婦で、子どもがいる。
5. 全体的に淡く明るい色を基調とし、写実的でない洗練されたタッチのイラストを使う。

## 3. AIが補足する重要事実

1. キャッチコピーはタイトルより目立たないが、タイトルを引き立てるように配置する。
2. キャッチコピーは下部に配置し、タイトルに次ぐ大きさにする。
3. 重要なキーワードを強調表示する。
4. 依頼者は写実的な人物表現を避け、イラスト調のビジュアルを好む。
5. 親しみやすさ・安心感を出すデザインが適切と考えられる。

## 4. 重要事実の分類

### 顧客目的に関わる事実

- 電子書籍の表紙デザイン案を募集している。
- メインタイトルを目立たせ、タイトルの魅力を伝えたい。
- キャッチコピーはタイトルを引き立てる役割を持つ。

### ターゲットに関わる事実

- 家族持ちの主婦がターゲット。
- 子どもがいる層を想定している。
- 日常の発信や副業に関心がある可能性が高い。

### 制約条件に関わる事実

- サイズ: 1,600 x 2,560 ピクセル。
- ファイル形式: JPEG。
- 著者名は不要。

### 必須文言に関わる事実

- メインタイトル: 「スマホ１本！超やさしい動画編集術」
- キャッチコピー: 「子どもの成長記録や趣味の発信に」「ちょっとした副業にもなる！」

### デザインに影響する事実

- 淡く明るい色を基調としたデザインを好む。
- 写実的な人物ではなく、適度に洗練されたタッチのイラストを希望している。
- キャッチコピーを下部に配置するレイアウトが望ましい。
- 重要なキーワードの強調表示が求められている。

## 5. なぜ重要か

### 顧客目的に関わる事実

事実: メインタイトルを最も目立つ形にする。
なぜ重要か: 表紙の第一印象で読者の関心を引き、内容の主旨を瞬時に伝えるため。

### ターゲットに関わる事実

事実: ターゲットは家族持ちの主婦、子どもがいる。
なぜ重要か: デザインや言葉遣い、色使いに安心感や親しみやすさを優先する必要がある。

### 制約条件に関わる事実

事実: サイズと形式が明確に指定されている。
なぜ重要か: 制作段階での納品ミスを防ぎ、実用的な表紙を作るため。

### 必須文言に関わる事実

事実: 特定のメインタイトルとキャッチコピーが指定されている。
なぜ重要か: 表紙の訴求力を決める主要な要素であり、見せ方が全体の印象を左右するため。

### デザインに影響する事実

事実: 淡く明るい色調、洗練されたイラスト、キャッチコピー下部配置、キーワード強調が好まれる。
なぜ重要か: 顧客の好みと訴求対象の印象を合わせ、採用されやすい表紙案にするため。

## 6. 事実・推論・確認事項の整理

### 事実

- サイズは1,600 x 2,560。
- 形式はJPEG。
- 著者名は不要。
- メインタイトル「スマホ１本！超やさしい動画編集術」が指定されている。
- ターゲットは家族持ちの主婦、子どもがいる。
- 淡く明るい色、洗練されたイラスト風の表現が好まれる。
- キャッチコピーは下部に配置しタイトルを引き立てる。
- 重要なキーワードを強調する。

### 推論

- ターゲットが共感しやすいのは、家庭や子育ての安心感を感じるビジュアル。
- キャッチコピーで「成長記録」「趣味」「副業」の価値をやさしく伝えると効果的。
- 表紙全体は親しみやすさと信頼感を両立させるとよい。

### 確認事項

- キャッチコピーの正式な配置／改行ルールはどれか。
- 色の指定や避けたい色の有無。
- イラストの具体的なモチーフやタッチの好み（例：手描き風、フラット、パステル調など）。

## 7. この時点で判断してよいこと

- メインタイトルを最も目立たせるレイアウトにする。
- ターゲットに親しみやすく安心感を与える配色を選ぶ。
- キャッチコピーはタイトルに次ぐ大きさで下部に配置する。
- 重要なキーワードを強調表示する。

## 8. この時点では判断してはいけないこと

- 正確なイラスト画風の細部や人物キャラクターのデザイン。
- どの色だけを必ず使うか。
- 表紙全体の最終レイアウトの細部。

## 9. 学び

案件文を読み解く際は、まず「目的・ターゲット・制約・必須要素」を明確に整理することが重要である。デザイン案を進める前に、これらの事実を事前に分類しておくことで、顧客の要望から外れにくい制作ができる。
