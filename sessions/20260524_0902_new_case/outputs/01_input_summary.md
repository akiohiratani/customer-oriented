# 01_input_summary

## 依頼内容概要

電子書籍の表紙デザイン案募集。

### 条件

- サイズ: 1,600 x 2,560 ピクセル
- ファイル形式: JPEG
- 著者名: 今回は不要
- メインタイトル: 「スマホ１本！超やさしい動画編集術」
- キャッチコピー: メインタイトルより目立たないが、タイトルを引き立てる配置

### ターゲット

- 家族持ちの主婦
- 子どもがいる

### 依頼者の好み・デザイン傾向

- 全体的に淡く明るい色を基調としたデザイン
- 写実的な人物ではなく、適度に洗練されたタッチのイラスト
- キャッチコピーは下部に配置し、タイトルを引き立てる大きさにする
- 重要なキーワードは強調表示する

## 注意点

- 著者名は表紙に入れない
- メインタイトルを最も目立たせる
- キャッチコピーはタイトルより控えめだが引き立てる役割にする
